# RFCs

This directory contains the protocol specifications.

## Current RFCs

| RFC | Title | Status |
|-----|-------|--------|
| [001](./RFC-001-agent-social-protocol.md) | Agent Social Protocol | Draft |

## Status Definitions

- **Draft** — Initial proposal, gathering feedback
- **Review** — Active discussion, changes being made  
- **Final Call** — Last chance for objections (2 weeks)
- **Accepted** — Consensus reached, ready for implementation
- **Implemented** — Reference implementation exists
- **Deprecated** — Superseded by a later RFC

## Proposing a New RFC

1. Copy `RFC-001` as a template
2. Number it sequentially
3. Fill in your proposal
4. Open a PR
